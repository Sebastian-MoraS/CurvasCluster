# Curvas Cluster

El repositorio se puede dividir en 2 grupos, los que sirven de ayuda para filtros - graficos (los archivos ElbowMethod.py, GraphicsHelper.py, helpers.py), en donde todo se junta en el archivo CurvasEstudio, donde se grafican las curvas filtradas a través de las librerias de scikit - learn. Todavia subo los archivos con los datos utilizados en el estudio, estoy viendo donde puedo subirlos quizas por medio de google drive u otro medio como Google colab.

El código CurvasEstudio, toma una serie de datos de lo podrian ser medidores inteligentes (MI), estos se filtran según ciertos criterios, esto es para obtener las mejores curvas para obtener el mejor tipo de representación posible, esto es necesario para poder escalar correctamente la carga en simulaciones de cargas en flujos de potencia.
